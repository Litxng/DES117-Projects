Comparative Analysis

July 21st, 2025

For my interactive narrative project, I am working on a website called Message in a Bottle. The idea is to recreate messages in bottles that float on the ocean. People can click on a bottle, read anonymous messages, and either reply or toss the bottle back into the sea. It is meant to be like a little ocean of thoughts.

Of course, I had to do some research and see if there are any websites that already exists with the same idea. Fortunately, I couldn’t find anything similar, but I did find this wonderful website that has a similar UI concept that I am going for.

LA Rosee Paris

link to site

The website is in French, so I couldn’t fully understand what the website is about. However, I am assuming that it is a website that shows off their product.

What makes this website unique is that it is interactive. The background is 3D graphics of their store, and you can click on each item to learn more about it.

What I like:

There is a lot of stuff on the screen, yet it feels minimal since it is clearly indicted with the dots that there are only 3 things you can click on

The website is fully interactive and immerse. It is like I am exploring the actual store but digitally

It feels like I am going on a journey. It has a story telling aspect where I have to explore everything on one section before unlocking the next

The website is so good that I honestly won’t change anything about it. The color theme is great. It is responsive and doesn’t take that long to load.

I also plan to do have the same UI where user would click something and a description of it will pop up. My original idea was to have the pop up cover up the whole screen but I kind of like the side by side view that LA Rosee has. I will incorporate that into my website and hopefully my website will look as great as theirs!
