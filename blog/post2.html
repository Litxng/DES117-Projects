Converting a Memo into a Museum-Themed Webpage

For a recent HTML exercise, I turned a rhetorical analysis memo about the Davis Design Museum into a simple website article. Since we were limited to using only basic CSS — no layout formatting — I focused on experimenting with fonts, colors, and borders.

Color Choices

For the color theme, I chose a beige background paired with brown text. Beige felt soft, easy on the eyes, and fitting for a museum atmosphere. Brown added warmth and contrast while staying within a cohesive palette. These subtle choices helped create a calm, thoughtful tone that matched the subject matter.

Playing with Figures and Captions

One of the most interesting things I learned was how to use the <figure> and <figurecaption> tags. Before this, I had only used <img> for pictures. These new tags helped me organize my images better and give them context. It made me realize how much more I need to explore in HTML.

What’s Next

This project also made me think about how different writing for the web is compared to writing a paper. On a website, content needs to be broken up clearly, with readable text and visual cues. With limited CSS tools, I learned to rely more on thoughtful design choices rather than flashy features.

Overall, this assignment gave me a better understanding of how design and content work together — even with basic tools.
